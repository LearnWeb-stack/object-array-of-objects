JavaScript Objects Tutorial

A Complete Guide for High School Students

1. What are Objects?

An object is a special data type that allows you to store multiple related values together as properties. Think of it like a real-world object that has characteristics.

Real-world example:

  • A car has properties: brand, model, color, year
  • A student has properties: name, age, grade, school
  • A book has properties: title, author, pages, price

In JavaScript, an object groups related data together using key-value pairs.

// Example of a student object let student = { name: "Sarah", age: 16, grade: 11, school: "Lincoln High" };

2. Why We Need Objects?

Objects help us organize related information in a clean and meaningful way.

Without Objects (messy):

let studentName = "John"; let studentAge = 17; let studentGrade = 12; let studentSchool = "Oak High"; let studentName2 = "Emma"; let studentAge2 = 16; let studentGrade2 = 11; let studentSchool2 = "Oak High";

This gets very messy with multiple students!

With Objects (organized):

let student1 = { name: "John", age: 17, grade: 12, school: "Oak High" }; let student2 = { name: "Emma", age: 16, grade: 11, school: "Oak High" };

Benefits:

  • Groups related data together
  • Makes code easier to read and understand
  • Easier to pass around in your program
  • Better represents real-world entities

3. How to Declare an Object

Syntax:

let objectName = { property1: value1, property2: value2, property3: value3 };

Objects are created using curly braces { }. Inside, we define properties using the format key: value, separated by commas.

Example 1: Creating a car object

let car = { brand: "Toyota", model: "Camry", year: 2022, color: "blue" };

Example 2: Creating a book object

let book = { title: "The Great Gatsby", author: "F. Scott Fitzgerald", pages: 180, published: 1925 };

Example 3: Creating a product object

let product = { name: "Laptop", price: 899.99, inStock: true, category: "Electronics" };
Note: Property names (keys) are on the left, values are on the right. Values can be strings, numbers, booleans, etc.

4. Accessing Object Properties (Dot Notation)

Syntax:

objectName.propertyName

To access a specific property of an object, use the dot notation: write the object name, followed by a dot, followed by the property name.

Example:

let student = { name: "Mike", age: 17, grade: 12 }; console.log(student.name); // Output: Mike console.log(student.age); // Output: 17 console.log(student.grade); // Output: 12

Using Object Properties:

let car = { brand: "Honda", model: "Civic", year: 2023 }; let message = "I drive a " + car.brand + " " + car.model; console.log(message); // Output: I drive a Honda Civic

Modifying Object Properties:

let person = { name: "Lisa", age: 25, city: "Seattle" }; person.age = 26; // Change age person.city = "Portland"; // Change city console.log(person.age); // Output: 26 console.log(person.city); // Output: Portland

5. Array of Objects

An array of objects is simply an array where each element is an object. This is very useful when you have multiple items of the same type.

Syntax:

let arrayName = [ { property1: value1, property2: value2 }, { property1: value1, property2: value2 }, { property1: value1, property2: value2 } ];

Example 1: Array of students

let students = [ { name: "John", age: 16, grade: 10 }, { name: "Sarah", age: 17, grade: 11 }, { name: "Mike", age: 16, grade: 10 }, { name: "Emma", age: 18, grade: 12 } ];

Example 2: Array of products

let products = [ { name: "Laptop", price: 999, category: "Electronics" }, { name: "Desk", price: 299, category: "Furniture" }, { name: "Mouse", price: 25, category: "Electronics" } ];

Example 3: Array of books

let books = [ { title: "1984", author: "George Orwell", pages: 328 }, { title: "To Kill a Mockingbird", author: "Harper Lee", pages: 281 }, { title: "The Hobbit", author: "J.R.R. Tolkien", pages: 310 } ];
Why use arrays of objects? When you have multiple similar items (like multiple students, products, or books), arrays of objects keep your data organized and easy to work with.

6. Accessing Array of Objects

To access a specific object in an array, use the array index. Then use dot notation to access properties of that object.

Syntax:

arrayName[index].propertyName

Step-by-step breakdown:

  1. arrayName[index] - Gets the object at that index
  2. .propertyName - Gets the property from that object

Example:

let students = [ { name: "Alex", age: 16, grade: 10 }, { name: "Beth", age: 17, grade: 11 }, { name: "Carl", age: 16, grade: 10 } ]; // Access the first student (index 0) console.log(students[0].name); // Output: Alex console.log(students[0].age); // Output: 16 // Access the second student (index 1) console.log(students[1].name); // Output: Beth console.log(students[1].grade); // Output: 11 // Access the third student (index 2) console.log(students[2].name); // Output: Carl

Another Example:

let products = [ { name: "Phone", price: 699 }, { name: "Tablet", price: 499 }, { name: "Laptop", price: 1299 } ]; console.log(products[0].name); // Output: Phone console.log(products[0].price); // Output: 699 console.log(products[2].name); // Output: Laptop console.log(products[2].price); // Output: 1299

7. Displaying Array of Objects Using Classic for Loop

Syntax:

for (let i = 0; i < arrayName.length; i++) { console.log(arrayName[i].propertyName); }

The classic for loop uses an index to access each object in the array, then uses dot notation to access properties.

Example 1: Display student names

let students = [ { name: "Alice", age: 16, grade: 10 }, { name: "Bob", age: 17, grade: 11 }, { name: "Charlie", age: 16, grade: 10 } ]; for (let i = 0; i < students.length; i++) { console.log(students[i].name); } // Output: // Alice // Bob // Charlie

Example 2: Display all properties

let products = [ { name: "Keyboard", price: 79 }, { name: "Monitor", price: 299 }, { name: "Mouse", price: 25 } ]; for (let i = 0; i < products.length; i++) { console.log("Product: " + products[i].name); console.log("Price: $" + products[i].price); console.log("---"); } // Output: // Product: Keyboard // Price: $79 // --- // Product: Monitor // Price: $299 // --- // Product: Mouse // Price: $25 // ---

Example 3: Using index in output

let cars = [ { brand: "Toyota", model: "Camry" }, { brand: "Honda", model: "Civic" }, { brand: "Ford", model: "Fusion" } ]; for (let i = 0; i < cars.length; i++) { console.log("Car " + (i + 1) + ": " + cars[i].brand + " " + cars[i].model); } // Output: // Car 1: Toyota Camry // Car 2: Honda Civic // Car 3: Ford Fusion
Why use classic for loop? When you need the index number for numbering or positioning items.

8. Displaying Array of Objects Using for...of Loop

Syntax:

for (let object of arrayName) { console.log(object.propertyName); }

The for...of loop gives you each object directly, making it simpler when you don't need the index.

Example 1: Display book titles

let books = [ { title: "Harry Potter", author: "J.K. Rowling" }, { title: "Lord of the Rings", author: "J.R.R. Tolkien" }, { title: "1984", author: "George Orwell" } ]; for (let book of books) { console.log(book.title); } // Output: // Harry Potter // Lord of the Rings // 1984

Example 2: Display multiple properties

let students = [ { name: "David", age: 17, grade: 11 }, { name: "Emma", age: 16, grade: 10 }, { name: "Frank", age: 18, grade: 12 } ]; for (let student of students) { console.log(student.name + " is " + student.age + " years old"); } // Output: // David is 17 years old // Emma is 16 years old // Frank is 18 years old

Example 3: Calculate total

let items = [ { name: "Shirt", price: 29.99 }, { name: "Pants", price: 49.99 }, { name: "Shoes", price: 79.99 } ]; let total = 0; for (let item of items) { console.log(item.name + ": $" + item.price); total += item.price; } console.log("Total: $" + total); // Output: // Shirt: $29.99 // Pants: $49.99 // Shoes: $79.99 // Total: $159.97
Why use for...of? Cleaner and simpler when you just need to access each object without needing the index number.

9. Displaying Array of Objects Using forEach()

Syntax:

arrayName.forEach(function(object, index) { console.log(object.propertyName); });

The forEach() method executes a function for each object in the array. It gives you both the object and its index.

Parameters:

  • object - The current object being processed
  • index - The index of the current object (optional)

Example 1: Display using only object

let movies = [ { title: "Inception", year: 2010 }, { title: "The Matrix", year: 1999 }, { title: "Interstellar", year: 2014 } ]; movies.forEach(function(movie) { console.log(movie.title + " (" + movie.year + ")"); }); // Output: // Inception (2010) // The Matrix (1999) // Interstellar (2014)

Example 2: Display using object and index

let games = [ { name: "Chess", players: 2 }, { name: "Soccer", players: 22 }, { name: "Tennis", players: 2 } ]; games.forEach(function(game, index) { console.log((index + 1) + ". " + game.name + " - " + game.players + " players"); }); // Output: // 1. Chess - 2 players // 2. Soccer - 22 players // 3. Tennis - 2 players

Example 3: Perform calculations

let scores = [ { student: "John", score: 85 }, { student: "Sarah", score: 92 }, { student: "Mike", score: 78 } ]; let sum = 0; scores.forEach(function(item) { console.log(item.student + " scored " + item.score); sum += item.score; }); console.log("Average: " + (sum / scores.length)); // Output: // John scored 85 // Sarah scored 92 // Mike scored 78 // Average: 85
Why use forEach()? Modern and clean syntax. Great when you want to perform an action on each object. Can use both the object and index when needed.

10. Comparison of Loop Methods

Quick Comparison Table

Loop Type Access Index? Syntax Complexity Best Used When
Classic for Yes Medium You need the index number
for...of No Simple You only need the object values
forEach() Yes (optional) Simple Modern code, optional index access

Same Task - Three Different Ways

Let's display the same data using all three methods:

let phones = [ { brand: "Apple", model: "iPhone 14" }, { brand: "Samsung", model: "Galaxy S23" }, { brand: "Google", model: "Pixel 7" } ]; // Method 1: Classic for loop console.log("Using classic for loop:"); for (let i = 0; i < phones.length; i++) { console.log(phones[i].brand + " " + phones[i].model); } // Method 2: for...of loop console.log("\nUsing for...of loop:"); for (let phone of phones) { console.log(phone.brand + " " + phone.model); } // Method 3: forEach() console.log("\nUsing forEach():"); phones.forEach(function(phone) { console.log(phone.brand + " " + phone.model); });

All three produce the same output:

Apple iPhone 14 Samsung Galaxy S23 Google Pixel 7
